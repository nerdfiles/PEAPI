# PEAPI

Keep track of your proofofexistence.com records through a simple CLI.

## Install

    $ npm install PE -g

## Use

    $ pe logs

      (hash) : (date) : (status)
      (hash) : (date) : (status)
      (hash) : (date) : (status)
      ...
      (< prev) (1) 2 3 4 5 ... (next >)

    $ pe check <file>

      (status)

    $ pe reg <file>

      (address)
